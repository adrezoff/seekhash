[Application.java](src/main/java/ru/adrezoff/cli/application/Application.java) — точка входа.

## Синтаксис
- Знак `>` указывает на поле ввода команды и её аргументов (если они есть). 
- В описании команд аргументы в `<` `>` скобках являются **обязательными**.
- В описании команд аргументы в `[` `]` скобках являются **НЕ обязательными**.
- Знак `|` отвечает за варианты аргумента (_или_). Например: `> exapmle_command <arg1|arg2|arg3>`
- 

## Команды
```
help [command] — все доступные команды, с аргументом [command] выведется подробная информация о команде.
exit — завершение с использованием graceful shutdown.
```
