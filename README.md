# seekhash

Приложение для поиска прообраза результата хеш функций. Один экземпляр, запущенного приложения, может искать только один прообраз. 

О проекте
--- 
- Автор: [adrezoff](https://github.com/adrezoff)
- Цель: выявить неэффективные хеш-функции.

## Ветки

`master` — релизная ветка, от неё создаются теги версий.

`develop` — ветка разработки в неё вливаются фича-ветки.

Новый релиз — это слияние из `develop` в `master`.

## Технологии

- Java 20
- Maven

## Структура

- `cli-application` — точка входа, консольное приложение. ([cli-application/README.md](cli-application/README.md))
